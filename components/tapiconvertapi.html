It converts file to another type of file. It utilizes the [WhoMeta Convert API](https://docs.totaljs.com/totalapi/6c221003eo51c/#13ff8001db51d). __Input data:__ ```js // {String} dataURI of a file "data:application/pdf;base64,JVBERi0xLjcN..." ``` __Output data__: Returns data according to the component configuration `Response type`. It can be `Buffer`, `Base64` or `DataURI`.
