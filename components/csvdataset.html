 The component stores data as CSV Dataset for e.g. Dashboard application. Dataset is stored in `/private/` path that is available with the help of the Flow access token. CSV file contains only processed values (according to the period) and the `.meta` file contains meta data + last value. __Input for data processing__: ```js { key: String, // optional, e.g. device_id | device_name value: Number // measured value } ``` ### CSV columns - `a` = `key {String}` - `b` = `value {Number}` - `c` = `min {Number}` - `d` = `max {Number}` - `e` = `sum {Number}` - `f` = `count {Number}` - `g` = `tshour {Number}`in the form `yyyyMMddhh` - `h` = `tsday {Number}` in the form `yyyyMMdd` - `i` = `tsweek {Number}` in the form `yyyyww` - `j` = `tsmonth {Number}` in the form `yyyyMM` - `k` = `tsyear {Number}` in the form `yyyy`
