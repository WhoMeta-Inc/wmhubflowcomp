OCR reader for `JPG`, `PNG` or `PDF` files. It utilizes the [WhoMeta OCR API](https://help.whometa.io/totalapi/6c221003eo51c/#If83x81cy61f). __Input data:__ ```js // {String} dataURI of a file "data:image/png;base64,iVBORw0KGgo....." ``` __Output data:__ ```js { text: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do...' } ```

Get WhoMeta API token.
The token field supports variables in the form {variable}.
