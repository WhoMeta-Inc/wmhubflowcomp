## Time-Based One-Time Password Algorithm This component can generate a secret and verifies the code (generated via Authenticator app) as a one-time password for 2FA authorization. - recommended mobile apps: __Google Authenticator__ or __Microsoft Authenticator__ --- __How does it work?__ First, a user must generate a secret via`generate` input in the authenticator app. Then, for verification of the one-time generated password, you need to use the second input `verify,` which needs user secrets. ### `generate` input/output __Input data__: - `user {String}` email or user/account name without special characters __Output data `object` with__: - `secret {String}` a generated secret which you must store for the specific user - `url {String}` an optimized url address for tokenization - `qrcode {String}` a link to the picture with QR code ### `verify` input/outputs __Input data `object` with__: - `secret {String}` __Output data `object` with__: - `success {Boolean}` determines if the password is valid or not
