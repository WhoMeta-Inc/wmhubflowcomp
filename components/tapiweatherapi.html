By using this component, you can obtain the current weather by an address. It utilizes the [WhoMeta Weather API](https://help.whometa.io/totalapi/6c221003eo51c/#128f7001kc51d). __Input data:__ ```js // {String} address without diacritics 'Banska Bystrica, Slovakia' ``` __Output data__: ```js { name: 'Bratislava', temperature: 22, wind: 20.2, humidity: 65, pressure: 1018, cloud: 25, uv: 6, sky: 'Partly cloudy' } ```
